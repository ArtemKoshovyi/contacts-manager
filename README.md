# Contacts-manager

Prosty menedżer kontaktów oparty na Django.

## Jak uruchomić projekt

1. Pobierz projekt (Code → Download ZIP)  
2. Rozpakuj archiwum  
3. Przejdź do folderu projektu
4. Otwórz terminal
5. Zainstaluj zależności:
   ```bash
   pip install -r requirements.txt
   
6. Uruchom serwer:

   ```bash
   python manage.py runserver
   
Otwórz w przeglądarce:
http://127.0.0.1:8000/
